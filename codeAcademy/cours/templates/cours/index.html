Grade Analyzer

Ask the user how many students are in the class. Then collect each student's grade and display the highest grade, lowest grade, and class average.

Example:

How many students? 4
Enter grade for student 1: 12
Enter grade for student 2: 15
Enter grade for student 3: 8
Enter grade for student 4: 10

Highest grade: 15
Lowest grade: 8
Class average: 11.25

💡 Use a list to store grades, and use max(), min(), and sum().

grades = []
for i in range(num_students):
    grade = float(input(f"Enter grade for student {i+1}: "))
    grades.append(grade)

print(f"Highest grade: {max(grades)}")
print(f"Lowest grade: {min(grades)}")
print(f"Class average: {sum(grades)/len(grades)}")

Even or Odd Counter

Ask the user to enter numbers one by one. When they type "stop", display how many were even and how many were odd.

Example:

Enter a number (or 'stop'): 3
Enter a number (or 'stop'): 12
Enter a number (or 'stop'): 7
Enter a number (or 'stop'): stop

Even numbers: 1
Odd numbers: 2

💡 Use while and check each number with %.

even_count = 0
odd_count = 0

while True:
    num = input("Enter a number (or 'stop'): ")
    if num == 'stop':
        break
    
    if int(num) % 2 == 0:
        even_count += 1
    else:
        odd_count += 1

print(f"\nEven numbers: {even_count}")
print(f"Odd numbers: {odd_count}")

Count the Vowels

Ask the user to input a sentence and count how many vowels it contains.

Example:

Enter a sentence: I love Python!
Number of vowels: 4

💡 Use a loop and .lower() to check for vowels.

sentence = input("Enter a sentence: ")
vowels = 'aeiou'
count = sum(1 for char in sentence.lower() if char in vowels)
print(f"Number of vowels: {count}")

Multiply Table Generator

Ask the user to enter a number and print the multiplication table from 1 to 10.

Example:

Reverse Words

Ask the user to enter a sentence and display it with the words in reverse order.

Example:

Enter a sentence: Python is fun
Reversed: fun is Python

💡 Use .split(), [::-1], and " ".join().

sentence = input("Enter a sentence: ")
words = sentence.split()
reversed_sentence = " ".join(words[::-1])
print(f"Reversed: {reversed_sentence}")

Password Validator

Check if a password has at least 6 characters, one digit, and one uppercase letter.

Example:

Enter password: Hello1
✅ Password is valid!

Enter password: hello
❌ Password is invalid.

💡 Use any(), isdigit(), and isupper().

password = input("Enter password: ")
is_valid = (
    len(password) >= 6 and
    any(c.isdigit() for c in password) and
    any(c.isupper() for c in password)
)
print("✅ Password is valid!" if is_valid else "❌ Password is invalid.")

First and Last Letter Game

Ask for a word and check if its first and last letters are the same (case-insensitive).

Example:

Enter a word: radar

💡 Compare word[0].lower() and word[-1].lower().

word = input("Enter a word: ")
if word[0].lower() == word[-1].lower():
